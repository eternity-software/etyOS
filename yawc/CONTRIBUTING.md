# Contributing to YAWC

Thanks for helping build YAWC as part of the future `etyOS` / `etyDE` stack.

## Development Flow

1. Build the project:

   ```bash
   ./scripts/build.sh
   ```

2. Run the compositor:

   ```bash
   ./scripts/run.sh
   ```

3. Run the smoke test launcher:

   ```bash
   ./scripts/run-test.sh
   ```

## Reporting Changes

When contributing, please include:

- what changed
- why it changed
- how it was tested
- any known limitations or follow-up work

## Scope

YAWC is evolving into a system compositor, not just a demo app.
Please prefer changes that keep the code modular, real, and maintainable.
